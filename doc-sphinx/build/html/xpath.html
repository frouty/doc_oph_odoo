Xpath¶

Indispensable pour modifier une vue existente : inherit

dans eclipse / market place / xpath / search . Installer le module. Restart eclipse.

Comment trouver la vue que l’on veut editer?¶

On se met en mode debug : settings / general settings / tout en bas Developer tools.

Ensuite sur la vue on clique sur l’icone de coccinel et on va en bas sur edit form view , tree view et on fait une recherce sur le code dans eclipse.

Comment trouver le xpath¶

Ensuite dans eclipse

Ouvrir fichier xml, ouvrir la perspective XML en haut à droite.

Window / show view / other / other / xpath

On met un xpath et on recarde ce qu’il retourne.

exemple d’xpath : https://www.w3schools.com/xml/xml_xpath.asp
